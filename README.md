# react-music-player
music player build with react, redux, react-router, webpack, ES6

### How to run
 - Install Dependencies
    ```shell
    npm install
    ```
 - Start the project
     ```shell
     npm start
     ```
 - Build the project 
     ```shell
     npm run build
     ```
 - [Demo](https://miraclehu.github.io/react-music-player/)
